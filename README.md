2. Self joins. Common in product.
3. DISTINCT and GROUP BY

6. SUM and COUNT
7. Date-time manipulation
8. String formatting, substring
9. Window functions like rank and row
10. Subqueries
11. HAVING vs WHERE
12. In some cases LAG and LEAD
13. Understanding indexing
14. Running totals
15. MAX and MIN
